# 03. デザイン指針

本ツールは、審査員が競技中にストレスなく、かつ正確に採点を行えるよう、Tailwind CSS を基盤としたモダンなデザインシステムを採用し、高い操作性と視覚的な一貫性を両立させる。

## 1. 基本方針

### 1.1. 視覚的印象（プレミアム & プロフェッショナル）
- **Slate システム**: ベースカラーに `Slate`（青みがかったグレー）系統を採用し、プロフェッショナルで落ち着いたインターフェースを提供する。
- **アクセントカラー**: アクションや重要な状態変更には `Primary`（活気のある青）を使用し、ユーザーの視線を適切に誘導する。
- **洗練された質感**: 背景のぼかし（`backdrop-blur`）やソフトな影（`shadow-md`）、角丸（`rounded-xl`）を効果的に活用し、プレミアムな外観を実現する。

### 1.2. 信頼性と視認性
- **等幅フォント**: 数値入力・表示箇所には `tabular-nums` を指定し、桁数の変動によるガタつきを抑え、視覚的なミスを防止する。
- **入力の最適化**: 
    - 数値入力時に「010」のような不要なゼロ埋め状態が保持されないよう、フロント側で自動的に先頭の0を取り除く（trim）処理を行う。
    - 入力欄にフォーカス（クリックまたはタブ移動）した際、内部のテキストを全選択状態にし、既存値の書き換えを容易にする。
- **厳格な入力補正**: 割合（%）による入力時は、算出されたスコアが「入力単位」の倍数になるようリアルタイムで補正を行う。
    - 最も近い適切な値に補正し、乖離が等しい場合は低い方に寄せる。
    - 補正結果は即座に割合入力欄にもフィードバック（同期）させ、不整合を排除する。
- **セマンティックカラー**: 状態（成功・警告・危険）に応じて標準化された色（`Success`, `Warning`, `Danger`）を使用し、情報の意味を直感的に伝える。
- **トップ3の強調**: 上位3名は専用のバッジやカラー（金・銀・銅）で強調し、順位の変動を際立たせる。4位以下は装飾（枠線や背景）のないプレーンなテキストで表示し、上位との視覚的な差を明確にする。

### 1.3. レイアウト指針（Tailwind CSS 基盤）
- **2カラム構成**: 大画面では「広大なメイン領域」と「操作・設定用のサイドパネル」に分割し、情報の視認性と操作の即時性を両立させる。サイドパネルは画面右端に配置する。
- **ユーティリティファースト**: インラインの独自スタイルを排除し、Tailwind CSS の標準クラス（Grid, Flex, Spacing）を用いてレイアウトを構築する。
- **レスポンシブ・アダプティブ**: `md:grid-cols-2` や `xl:col-span-3` 等のブレークポイントを活用し、画面幅に応じてサイドパネルをメインコンテンツの下部にスタックさせる等の柔軟な調整を行う。
- **インタラクティブな微細挙動**: ボタンのホバー（`hover:opacity-80`）やコンポーネントの出現アニメーション（`animate-in`）を付加し、システムの応答性を視覚的に伝える。
- **一貫した余白**: Spacing スケール（`gap-4`, `p-6` 等）を統一して使用し、画面全体の調和を保つ。
