# 03. デザイン指針

本ツールは、審査員が競技中にストレスなく、かつ正確に採点を行えるよう、モダンなUIと高い操作性を両立させたデザインを目指す。

## 1. デザイン指針

### 1.1. 視覚的印象（プレミアム & プロフェッショナル）
- **モダンな質感**: グラスモルフィズム（背景ぼかし）や洗練されたグラデーションを活用し、単なる事務ツールではない「競技会の熱量を高める」プレミアムな外観を提供する。
- **ダークモード重視**: 競技会場（暗い客席など）での使用を想定し、目に優しくコントラストの取れた配色を基本とする。

### 1.2. 信頼性と視認性
- **正確な数値表示**: 数値入力・表示箇所には等幅フォント（`tabular-nums`）を使用し、視覚的なミスを防止する。
- **入力の最適化**: 数値入力時に「010」のような不要なゼロ埋め状態が保持されないよう、フロント側で自動的に先頭の0を取り除く処理（trim）を行う。
- **即時反映**: 入力した値は直ちに合計値・順位・統計グラフに反映される。
- **トップ3の強調**: 上位3名はアイコンやカラー（金・銀・銅）で強調し、順位の変動を直感的に伝える。
